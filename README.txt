Script Names:
hw_Q2.txt -> hw_Q2.py
hw_Q3.txt -> hw_Q3.py
hw_Q4.txt -> hw_Q4.py

Hoe to Run hw_Q2.py:
-Install package nltk
pip install nltk
-Run the script
-Sentences will be printed on newlines


How to Run hw_Q3.py:
-Run the script
-Interactive prompt will appear
-Enter each character of an input string one at a time
-Prompt will respond with observed probabilities and predicted next characters
- Press Enter to exit


How to run hw_Q4.py:
-Install packages BeautifulSoup4 and Wikipedia
pip install wikipedia
pip install beautifulsoup4
-Run the script
-Output will appear in 'output.tsv'
